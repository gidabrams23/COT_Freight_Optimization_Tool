PJ Component Rendering Rules
Renderings are now assembled from interchangeable components: one deck body + one tongue profile.
Rules currently wired in load builder: Standard tongue = 4.0 ft, Gooseneck tongue = 9.0 ft. Deck length and deck height come from each SKU.
Tongue Components (Standalone)
Standard Tongue
C-channel profile. Fixed 4.0 ft render length. Attaches to either flat or dump deck body.
Gooseneck Tongue
Rising GN neck with over-cab coupler. Fixed 9.0 ft render length and larger ring coupler profile.
Deck Components (Standalone)
Flat Deck
Dual-axle flat body. Length and effective height come from the selected SKU dimensions.
Dump Deck
Open-top dump body. Closed rear by default; GN pairings render open rear gate to indicate neck insertion path.
Assembled Examples
Flat + Standard
Default flat-deck pairing with 4.0 ft tongue.
Flat + Gooseneck
Same flat deck with GN neck geometry and over-cab coupler.
Dump + Gooseneck
Dump body with open rear marker and GN coupler profile.
