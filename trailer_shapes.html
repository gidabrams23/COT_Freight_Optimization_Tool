RENDER CONTEXT These shapes appear inside unit cards on the load canvas. Each card's width is proportional to the unit's total_footprint. Each card's height is fixed (determined by the zone's height budget display). The SVG viewBox should be: "0 0 {total_footprint * scale_px_per_ft} 100" Use width="100%" height="100%" on the SVG so it fills the card. 2. PARAMETERIZATION PATTERN All x-coordinates that depend on bed length use: bed_px = bed_length_measured * scale_px_per_ft All x-coordinates that depend on tongue use: tongue_px = tongue_feet * scale_px_per_ft The SVG viewBox width = (bed_length_measured + tongue_feet) * scale_px_per_ft Vertical positions are fixed (do not scale with dimensions). 3. COLOR APPLICATION Each category has a stroke color and a fill color (stroke at 10-20% opacity). Apply stroke to all structural lines. Apply fill at rgba(r,g,b,0.10) to bed rectangles and neck shape. Apply fill at rgba(r,g,b,0.22-0.30) to tongue and accent elements. Wheel hubs: rgba(r,g,b,0.30) fill. Category → hex stroke color: utility: #0ea5e9 car_hauler, tilt: #22c55e dump_*: #f59e0b deck_over, tilt_deckover: #a78bfa gooseneck_*: #6366f1 4. VIOLATION STATE When the unit has a constraint violation, override the stroke color to a muted amber so the warning remains visible without overpowering the schematic linework. 5. NESTED UNIT (D5 inside dump) When is_nested=True on a D5 unit, render the D5 shape at 60% opacity positioned inside the host dump's box rectangle, centered horizontally, aligned to the bottom of the box interior. 6. GN AXLE DROP TOGGLE When gn_axle_dropped=True, add a small downward-arrow indicator (▼) near the rear axle pair to signal the axles are in the dropped position. Adjust coupler_y by +4px (coupler sits slightly lower when axles dropped). 7. LAYER CONTEXT Pass is_top_layer to the macro. When True: - Render spare mount stub on car_hauler shapes (it's normally hidden under next layer) - Use stack_height_top value for the height annotation When False: - Hide spare mount stub (covered by the unit stacked on top) - Use stack_height_mid value ============================================================ JINJA2 MACRO IMPLEMENTATION
